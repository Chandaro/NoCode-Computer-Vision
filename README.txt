║              NoCode CV Trainer  – Quick Start            ║
╚══════════════════════════════════════════════════════════╝

──────────────────────────────────────────────────────────
  FIRST TIME?   →  Double-click  "Install NoCode CV.bat"
──────────────────────────────────────────────────────────
  The installer will:
    1. Check your system (Python, Node.js, GPU/CUDA)
    2. Create an isolated Python environment (venv/)
    3. Install all backend & frontend dependencies
    4. Build the app
    5. Create a desktop shortcut for you

  Takes about 5–15 minutes depending on internet speed.

──────────────────────────────────────────────────────────
  ALREADY INSTALLED?   →  Double-click  "NoCode CV.bat"
                         or use the Desktop shortcut
──────────────────────────────────────────────────────────
  The app opens in your browser at  http://localhost:8000
  A small status window lets you stop the server.

──────────────────────────────────────────────────────────
  REQUIREMENTS (install manually before first run)
──────────────────────────────────────────────────────────
  • Python 3.9+   →  https://www.python.org/downloads/
                     ✅ Check "Add Python to PATH"

  • Node.js 16+   →  https://nodejs.org/en/download/

  • CUDA Toolkit  →  https://developer.nvidia.com/cuda-downloads
    (optional — only needed for GPU-accelerated training)
    Recommended: CUDA 12.4 or 12.8 for RTX 4000/5000 series
